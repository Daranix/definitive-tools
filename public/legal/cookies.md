# Cookie Policy

**Last Updated:** May 19, 2026

This Cookie Policy explains how **Definitive Tools** uses cookies and similar technologies. 

Our source repository is publicly hosted on [GitHub](https://github.com/Daranix/definitive-tools).

## 1. What Are Cookies?

Cookies are small text files stored on your computer or mobile device when you visit a website. They are widely used to make websites work, or work more efficiently, as well as to provide reporting information.

## 2. Do We Use Cookies?

**No.** Definitive Tools does not use third-party tracking, profiling, or advertising cookies. 

We do not track your behavior across other websites, and we do not sell or share any user details.

## 3. Local Storage Preferences

Instead of server-side tracking cookies, we may utilize your browser's local storage capabilities (`localStorage`) to save state settings locally on your machine. This includes:
- Active editor color theme (light/dark mode)
- Layout sizing preferences for Monaco Editor or Swagger UI
- Cached configuration selections for tools

This storage remains completely within your browser and is not transmitted to our servers. You can clear this data at any time by clearing your browser cache.

## 4. Open-Source Hosting Infrastructure

Because the codebase is hosted and distributed via GitHub, any interactions with the GitHub platform or repository files may be subject to GitHub's own privacy and cookie policies. We do not host the active live web application on GitHub Pages, only the source repository.

More information about the source code structure is available on our GitHub repository.
