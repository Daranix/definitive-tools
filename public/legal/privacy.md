# Privacy Policy

**Last Updated:** May 19, 2026

Welcome to **Definitive Tools** (the "Service"). We are committed to protecting your privacy and ensuring a secure online experience. 

Definitive Tools is an open-source suite of client-side developer utilities. The source code is publicly audited and available on [GitHub](https://github.com/Daranix/definitive-tools).

## 1. No Data Collection (Local Processing)

The core philosophy of Definitive Tools is privacy-first:
- **Local Execution:** All processing (such as QR generation, background image removal, Base64 conversion, JWT parsing, and OpenAPI code generation) occurs directly inside your web browser.
- **No Servers:** We do not upload your files, images, input texts, credentials, or keys to any remote servers. Your private data never leaves your local machine.

## 2. Third-Party Analytics and CDNs

- **Source Code Repository:** Our codebase is hosted on GitHub. We do not integrate any remote usage analytics or server trackers.
- **External Web Libraries:** Certain browser features use client-side runtimes (like CheerpJ from Leaning Technologies for OpenAPI generation). These runtimes are loaded directly in your browser.

## 3. Browser Storage and Cookies

Definitive Tools may use browser mechanisms like `localStorage` or `sessionStorage` strictly to persist your local preferences (such as editor themes or input settings) to improve your experience. We do not use tracking or advertising cookies.

## 4. Open Source

Since our project is entirely open-source, you can audit, verify, and run the code yourself by visiting our repository on GitHub.
